# 文章管理模块测试指南

## 📋 测试前准备

### 1. 后端配置
确保后端已配置以下内容：

#### 菜单配置
需要在后端菜单系统中添加以下菜单项：

```json
{
  "name": "文章管理",
  "path": "/iras/article",
  "component": "/iras/article/index",
  "icon": "lucide:file-text",
  "order": 2,
  "parentId": "iras-menu-id"
}
```

#### 编辑页面路由（可选，不显示在菜单中）
```json
{
  "name": "文章编辑",
  "path": "/iras/article/edit",
  "component": "/iras/article/edit",
  "hideInMenu": true,
  "parentId": "article-menu-id"
}
```

### 2. API 接口
确保以下接口已实现：
- `POST /iras-api/api/v1/article/list` - 文章列表
- `POST /iras-api/api/v1/article/save` - 保存文章
- `DELETE /iras-api/api/v1/article/{id}` - 删除文章
- `POST /iras-api/api/v1/article/ai/format` - AI格式化
- `POST /iras-api/api/v1/upload/image` - 图片上传
- `POST /iras-api/api/v1/article/categories/list` - 文章分类列表

## 🐛 常见问题

### 1. 页面无法访问
**原因**：后端菜单未配置或权限不足
**解决**：检查后端菜单配置和用户权限

### 2. 文章列表为空
**原因**：后端接口未返回数据或接口地址错误
**解决**：检查浏览器控制台网络请求，确认接口地址和返回数据

### 3. 图片上传失败
**原因**：上传接口未实现或文件大小超限
**解决**：检查后端上传接口实现和文件大小限制

### 4. 富文本编辑器不显示
**原因**：WangEditor 依赖未安装或样式文件未加载
**解决**：确认依赖已安装，检查浏览器控制台错误信息

### 5. 保存时提示验证失败
**原因**：必填字段未填写
**解决**：确保标题和分类字段已填写
