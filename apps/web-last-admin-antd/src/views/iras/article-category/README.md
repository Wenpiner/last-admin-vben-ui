# 文章分类管理模块

## 📋 功能概述

文章分类管理模块提供了完整的文章分类信息管理功能，支持树形结构的分类管理，包括增删改查等基本操作。

## 🚀 主要功能

### 1. 文章分类列表显示
- ✅ 树形表格展示文章分类信息
- ✅ 显示分类基本信息（名称、级别、排序、描述等）
- ✅ 支持多级分类展示
- ✅ **分类级别彩色标签**：不同级别使用不同颜色的标签显示，更加直观
  - L1: 蓝色 (blue)
  - L2: 绿色 (green)
  - L3: 橙色 (orange)
  - L4: 紫色 (purple)
  - L5: 青色 (cyan)
  - L6: 品红 (magenta)
  - L7: 火山红 (volcano)
  - L8: 金色 (gold)
  - L9: 青柠 (lime)
  - L10: 极客蓝 (geekblue)

### 2. 文章分类搜索
- ✅ 按分类名称搜索
- ✅ 按分类级别筛选

### 3. 文章分类管理
- ✅ 新增文章分类
- ✅ 编辑文章分类信息
- ✅ 删除文章分类（有子分类时不允许删除）
- ✅ 支持父子分类关系

## 🔧 技术实现

### API接口
- `POST /iras-api/api/v1/article/categories/list` - 获取文章分类列表
- `POST /iras-api/api/v1/article/categories/save` - 保存文章分类（创建/更新）
- `DELETE /iras-api/api/v1/article/categories/{id}` - 删除文章分类

### 数据结构
```typescript
interface CategoryInfo {
  id?: number;
  name: string;            // 分类名称
  description?: string;    // 分类描述
  parentId?: number;       // 父分类ID
  order: number;           // 排序
  level: number;           // 分类级别
  children?: CategoryInfo[]; // 子分类列表
  createdAt?: string;      // 创建时间
  updatedAt?: string;      // 更新时间
}
```

### 特色功能：分类级别彩色标签
通过 `getLevelColorMap()` 函数定义了10个级别的颜色映射，在表格中使用 Ant Design Vue 的 `Tag` 组件渲染，
使不同级别的分类一目了然。

## 🎯 使用方式

### 1. 访问文章分类管理
导航到 `iRas 管理 > 文章分类管理` 页面

### 2. 查看文章分类列表
- 页面加载时自动显示所有文章分类的树形列表
- 不同级别的分类以不同颜色的标签显示
- 使用搜索表单筛选文章分类

### 3. 新增文章分类
1. 点击 "新增文章分类" 按钮
2. 填写分类信息（名称、级别、排序等）
3. 可选择上级分类（不选则为顶级分类）
4. 点击确认保存

### 4. 编辑文章分类
1. 点击分类行的 "编辑" 按钮
2. 修改分类信息
3. 点击确认保存

### 5. 删除文章分类
1. 点击分类行的 "删除" 按钮
2. 确认删除操作
3. 注意：如果该分类下有子分类，则不允许删除

## 🌍 国际化支持

支持中英文双语：
- 中文：`zh-CN/iras.json` 中的 `articleCategory` 配置
- 英文：`en-US/iras.json` 中的 `articleCategory` 配置

## ⚠️ 注意事项

1. **树形结构**：分类支持多级树形结构，通过 `parentId` 字段关联父分类
2. **删除保护**：删除分类前会检查是否有子分类，有子分类时不允许删除
3. **表单验证**：所有必填字段都有相应的验证规则
4. **默认值**：新增分类时级别默认为1，排序默认为0
5. **分页支持**：支持分页加载，提升大数据量时的性能

## 🎨 UI 特色
