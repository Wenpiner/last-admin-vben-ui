# 配置管理模块

## 📋 功能概述

配置管理模块提供了完整的系统配置信息管理功能，支持增删改查等基本操作。通过配置管理，可以动态调整系统参数，无需重启应用即可生效。

## 🚀 主要功能

### 1. 配置列表显示
- ✅ 分页表格展示配置信息
- ✅ 显示配置键、名称、分组、值、描述等信息
- ✅ 配置分组标签化显示
- ✅ 长文本内容自动截断并提供 Tooltip 查看

### 2. 配置搜索
- ✅ 按配置键搜索
- ✅ 按配置名称搜索
- ✅ 按配置分组搜索

### 3. 配置管理
- ✅ 新增配置
- ✅ 编辑配置信息
- ✅ 删除配置
- ✅ 配置键格式验证

## 🔧 技术实现

### API接口
- `POST /sys-api/configuration/list` - 获取配置列表
- `POST /sys-api/configuration/createOrUpdate` - 创建或更新配置
- `POST /sys-api/configuration/delete` - 删除配置

### 数据结构
```typescript
interface ConfigurationInfo {
  key: string;              // 配置键
  value: string;            // 配置值
  name: string;             // 配置名称
  group: string;            // 配置分组
  description?: string;     // 配置描述
  createdAt?: number;       // 创建时间
  updatedAt?: number;       // 更新时间
}
```

## 🎯 使用方式

### 1. 访问配置管理
导航到 `系统管理 > 配置管理` 页面

### 2. 查看配置列表
- 页面加载时自动显示所有配置的分页列表
- 使用搜索表单筛选配置

### 3. 新增配置
1. 点击 "新增配置" 按钮
2. 填写配置信息：
   - **配置键**：唯一标识，只能包含字母、数字、下划线、点号和连字符
   - **配置名称**：配置的显示名称
   - **配置分组**：用于分类管理配置
   - **配置值**：具体的配置内容，支持文本、JSON等格式
   - **配置描述**：可选，用于说明配置的用途
3. 点击确认保存

### 4. 编辑配置
1. 点击配置行的 "编辑" 按钮
2. 修改配置信息（配置键不可修改）
3. 点击确认保存

### 5. 删除配置
1. 点击配置行的 "删除" 按钮
2. 确认删除操作

## 🌍 国际化支持

支持中英文双语：
- 中文：`zh-CN/system.json` 中的 `configuration` 配置
- 英文：`en-US/system.json` 中的 `configuration` 配置

## ⚠️ 注意事项

1. **配置键唯一性**：配置键在系统中必须唯一，创建后不可修改
2. **配置键格式**：只能包含字母、数字、下划线、点号和连字符
3. **配置值长度**：配置值最大支持2000个字符
4. **删除确认**：删除配置前会弹出确认对话框，删除后不可恢复
5. **分页加载**：列表支持分页，默认每页20条记录

## 🔄 扩展建议

1. **配置类型**：添加配置类型字段（字符串、数字、布尔值、JSON等）
2. **配置验证**：根据配置类型进行值的格式验证
3. **配置导入导出**：支持批量导入导出配置
4. **配置历史**：记录配置变更历史
5. **配置权限**：不同角色对不同分组的配置有不同的操作权限
6. **配置生效**：配置修改后的实时生效机制
7. **配置备份**：定期备份重要配置
